# 🎓 Ensemble Machine Learning Studio

머신러닝 기반으로 재학생의 중도탈락 가능성을 예측하고 주요 영향 요인을 분석하는 웹 서비스


### 주요 기능
- 데이터 업로드
- 하이퍼파라미터 자동 탐색
- 모델 학습
- 예측 수행
- SHAP 기반 설명 가능 AI 분석

## 🛠 Tech Stack

| 분야 | 기술 |
|--------|--------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Flask |
| ML | Scikit-Learn, LightGBM, XGBoost |
| Data | Pandas, NumPy |
| Explainability | SHAP |
